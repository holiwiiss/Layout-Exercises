# Responsive Background Images

## Objetivo

Practicar el uso de **media queries** para cambiar recursos visuales dependiendo del tamaño de pantalla.

El ejercicio consiste en mostrar una **imagen de fondo diferente en desktop y mobile** utilizando únicamente **HTML y CSS**.

## Tecnologías

- HTML5
- CSS3
- Media Queries

## Detalles Técnicos

- Imagen de fondo aplicada mediante `background-image`.
- Cambio de imagen mediante media query.

## Cómo ver el proyecto

Abrir `index.html` en un navegador y redimensionar la ventana para observar el cambio de imagen.
