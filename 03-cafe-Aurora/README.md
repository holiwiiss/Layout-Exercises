# Cafe Aurora - Mobile Mockup

## Objetivo
Recrear la maquetación de una página web para la cafetería **Cafe Aurora** a partir de un diseño en Figma, enfocándose en **dispositivos móviles**.

## Tecnologías
- HTML5
- CSS3
- Diseño basado en mockup de Figma

## Detalles
- Diseño **responsivo solo para móvil**, no incluye versión de escritorio.
- Se practicó la correcta **estructura semántica del HTML** según el mockup.
- Incluye secciones típicas de un sitio de cafetería: portada, menú, información de contacto y pie de página.
- Énfasis en mantener la estética y distribución del diseño original de Figma.

## Cómo ver
Abrir `index.html` en el navegador.
