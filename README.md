# Layout Exercises

Repositorio con ejercicios de maquetación HTML y CSS realizados como prácticas personales.

## Contenido
1. `01-newspaper-layout` – Maquetación de un periódico (no responsive)
2. `02-wow-cards` – Cartas de estadísticas para un videojuego (no responsive)
3. `03-coffee-mockup` – Mockup de un sitio web de cafetería (solo móvil)

Cada carpeta incluye su propio README con más detalles sobre el ejercicio.
