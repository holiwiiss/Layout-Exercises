# Layout Exercises

Repositorio con ejercicios de maquetación HTML y CSS realizados como prácticas personales.

## Contenido
1. `01-newspaper-layout` – Maquetación de un periódico (no responsive)
2. `02-videogame-cards` – Cartas de estadísticas para un videojuego (no responsive)
3. `03-cafe-Aurora` – Mockup de un sitio web de cafetería (solo móvil)
4. `04-responsive-semantic-layout` – Maquetación de un sitio web responsive con uso de variables CSS.
5. `05-responsive-calculator` – Maquetación de una calcuuladora responsive SIN JAVASCRIPT.

Cada carpeta incluye su propio README con más detalles sobre el ejercicio.
