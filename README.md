# Layout Exercises

Repositorio con ejercicios de maquetación HTML y CSS realizados como prácticas personales.

## Contenido
1. `01_newspaper_layout` – Maquetación de un periódico (no responsive)
2. `02_videogame_cards` – Cartas de estadísticas para un videojuego (no responsive)
3. `03_cafe_aurora` – Mockup de un sitio web de cafetería (solo móvil)

Cada carpeta incluye su propio README con más detalles sobre el ejercicio.
