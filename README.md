# Layout Exercises

Repositorio con ejercicios de maquetación HTML y CSS realizados como prácticas personales, que muestran la progresión desde maquetación básica hasta layouts responsive avanzados con buenas prácticas de CSS.

## Contenido
1. `01-newspaper-layout` – Maquetación de un periódico sencillo (no responsive)
2. `02-videogame-cards` – Cartas de estadísticas para un videojuego (no responsive)
3. `03-cafe-aurora` – Mockup de un sitio web de cafetería enfocado en móvil
4. `04-responsive-semantic-layout` – Maquetación de un sitio web responsive con semántica HTML y uso de variables CSS (BEM incluido)
5. `05-responsive-calculator` – Maquetación de una calculadora responsive (sin JavaScript, BEM y variables CSS)
6. `06-responsive-dog-cards-grid` – Página de cards responsive (sin JavaScrip, BEM y variables CSS)

Cada carpeta incluye su propio README con más detalles sobre el ejercicio, tecnologías utilizadas, y cómo ver el proyecto.
